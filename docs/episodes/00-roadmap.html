00 - Mental model and topology

The complete architecture before touching commands: hospitals, regions, gateways, and the global hub.

Demo objective

This project simulates hospitals collaborating to train an intrusion detector on CIC-IDS2017 network-flow data. Each hospital keeps its local partition. Regions aggregate hospital models, and the global layer aggregates regional models.

Roadmap

  1. Prepare the environment, Kaggle CLI, and dependencies.
  2. Download and clean CIC-IDS2017 without committing raw data.
  3. Create six non-IID partitions with seed 123.
  4. Start SuperLinks, SuperNodes, and SuperExecs with Docker Compose.
  5. Run regional federations with hospitals as clients.
  6. Run the global federation with regional gateways as clients.
  7. Evaluate the global checkpoint and review privacy boundaries.
