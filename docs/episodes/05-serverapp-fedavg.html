05 - Regional ServerApp and FedAvg

How the regional server aggregates hospitals and writes regional checkpoints.

Shared entry point

The ServerApp reads level from the run config. This lets the same component serve a regional federation or the global federation.

Regional federation

Each region uses FedAvg with weighted_by_key='num-examples'. The clients are the hospitals connected to the regional SuperLink.
