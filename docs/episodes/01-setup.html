01 - Setup from scratch

How to prepare Python, dependencies, Kaggle credentials, and the main targets.

Prerequisites

  • Python 3.11 or another version compatible with Flower 1.29.
  • Docker Desktop or Docker Engine with Compose.
  • Kaggle CLI credentials stored outside the repository.
  • GitHub CLI only if you want to publish or configure Pages from the terminal.

Installation

python -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install -e .
Credentials

Do not put real tokens in Git. Use environment variables or ~/.kaggle/kaggle.json. The repository ignores data, checkpoints, and common credential files.

export KAGGLE_API_TOKEN="<kaggle_token>"
make data
make partition SEED=123

Main targets

Direct demo

Once data and partitions exist, make demo starts Docker, runs one global round with one regional round, evaluates, and refreshes the renders.

make demo
