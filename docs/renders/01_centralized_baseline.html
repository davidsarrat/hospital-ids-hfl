01 - Baselines and Evaluation Tables

This vignette executes lightweight inspection code for the available metric reports. Full centralized/local baseline training can be run from the Makefile, but the notebook does not silently launch expensive training. It reports which artifacts exist and plots the current hierarchical evaluation output.
