This notebook provides non-hierarchical reference points for the same PyTorch MLP used in the Flower demo.

  • Centralized MLP: trains on all simulated hospital train splits.
  • Local-only MLP: trains one independent model per hospital.

These baselines are for comparison. The main architecture demonstration remains hierarchical Flower FedAvg.
